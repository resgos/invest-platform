Как подключиться к БД

SQLite — файловая СУБД. БД — это один файл: instance/invest.db. Для подключения не нужен сервер или пароль.

Командная строка (на сервере)

cd ~/investplatform
sqlite3 instance/invest.db
# Далее SQL-запросы, например:
.tables
SELECT * FROM users;
.quit

GUI: DB Browser for SQLite

Скачайте sqlitebrowser.org.
Скопируйте instance/invest.db на локальный ПК через SCP:
scp deploy@ВАШ_IP:~/investplatform/instance/invest.db ./invest.db
Откройте файл в DB Browser.

Python-скрипт

import sqlite3
conn = sqlite3.connect('instance/invest.db')
cursor = conn.cursor()
cursor.execute('SELECT * FROM users')
for row in cursor.fetchall():
    print(row)
conn.close()
